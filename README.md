## Z shell (Zsh)
First install [Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh) and some 3rd plugins:
```sh
sh sh/setup_zsh.sh
```

then source the file `sh/.zshrc`:
```sh
source sh/.zshrc
```
